## Description

[nestjs-shargea](https://github.com/denishamann/nestjs-shargea) is the backend of shargea made with Nest.
Shargea is an expense manager app.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
